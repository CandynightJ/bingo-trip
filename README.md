# Bingo Trip
## Overview
This is a project made for Transit Time! 
This project's objective is that you don't literally die of boredom on your bus trip, and instead, you look at your surroundings and play this fun bingo game.

## How it works
When you press play, a random bingo board is generated. Each bingo tile has an emoji. At the bottom, you have a key so you know what each emoji represents. This method was chosen so it is more readable. You need to tap the tile when the event happens

## Scoring
1 line gives you 10 points
The whole board (bingo) gives you 50 points!

## Tools
This project was made with HTML, javascript and TailwindCSS
